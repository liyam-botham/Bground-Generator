git clone : Clones a repo from github

git status: status of the files present in the git hub cloned folder

git add 'filename': adds file to online repo

git commit : Commit changes  {"-m" to add message}

git push : push changes to git

git branch : shows branches

git branch "New branch name": create new branch

git checkout "branch name" : change to select branch
